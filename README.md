# Hiding outline in an accessible way

Based on [this article](https://jmperezperez.com/outline-focus-ring-a11y/).

# Install

```
npm i outline-none
```

# Usage

Somewhere in your client side code:

```js
require("outline-none");
```

or

```js
import "outline-none";
```
